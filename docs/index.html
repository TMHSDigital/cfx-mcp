CFX MCP

Read-only MCP server for CFX/FiveM development. Three tools, no API key required.

  • cfx_getNative - look up a FiveM/RedM native by name, partial name, or hex hash
  • cfx_queryServer - inspect a live FiveM server via its public /info.json and /players.json
  • cfx_searchReleases - search cfx.re community releases via the Discourse API
